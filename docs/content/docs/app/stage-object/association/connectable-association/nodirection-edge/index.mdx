## 介绍

在2025.4.21之前，所有的连线都是“有向边”，即只能从一个实体指向另一个实体。而在2025.4.21之后，我们引入了“无向边”，即不存在方向的双向连接。

但为了更加的一般化，无向边不仅可以只连接两个实体，还可以连接多个实体。

因此 “多源无向边” 这个概念被引入，而 “无向边” 只是 “多源无向边” 的一种特例。


<Callout>
在图论中，"无向边"是指连接两个顶点并且没有方向的边。然而，标准图论中并没有直接称为“多源无向边”的概念。不过，图论中确实存在一种可以连接多个顶点的结构，称为“无向超边”（undirected hyperedge），它属于超图（hypergraph）的范畴。

超图是一种推广的图结构，其中的边可以连接一个以上的顶点。因此，你可以认为“多源无向边”是超图理论中的一种表现形式，即无向超边。在超图中，无向超边可以连接任意数量的节点，而不仅仅是两个。
</Callout>
