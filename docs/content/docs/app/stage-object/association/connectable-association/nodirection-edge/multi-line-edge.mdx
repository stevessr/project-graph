## 多源无向边连线

通过秘籍键创建：选中两个及以上的可连接实体，连按三次 `=` 键，创建多源无向边。

## 编辑名称

双击边，即可给多源无向边添加文字标注，文字标注会显示在多源无向边的中心点。

## 移动中心点

当多源无向边连接了两个以上的实体时，可以移动中心点位置，鼠标直接拖动边即可。

## 颜色

选中多源无向边，点击工具栏即可更改颜色

## 箭头方向

选中多源无向边，点击工具栏的箭头按钮，即可设置箭头方向。具体分为以下三种情况：

1. 向内
2. 向外
3. 不显示箭头

## 特性

删除任意一个多源无向边连接的实体，该边都会自动被删除。

> 随着时间的推移和用户反馈，可能会有更多的特性、转换方法被加入。
