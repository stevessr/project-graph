## 版本历史

从2024年9月21日 至 11月16日，Project Graph从PyQt版本转成Tauri版本，逐渐实现全部功能，总共用时56天。1.0.0正式发布

| 重要版本号   | 重要内容与版本号含义                |
|----------|------------------------------------|
| 1.0.0    | 完成Pyqt版本的全部功能，增加Section框 |
| 1.1.0    | 逻辑节点的自动计算引擎                 |
| 1.2.0    | 图片节点                           |
| 1.3.0    | 文本节点：支持输入换行符换行          |
| 1.4.0    | 连线：设置颜色                    |
| 1.4.20   | 涂鸦                           |
| 1.4.40   | 文本节点：调整宽度强制换行       |
| 1.5.0    | 移除MiSans字体，安装包体积大幅度减小 |
| 1.6.0    | 连线接头位置可以调整，贝塞尔曲线优化    |

若要查看详细的版本内容，请访问 [GitHub Releases](https://github.com/LiRenTech/project-graph/releases)
